class librep.datasets.motionsense.MotionSenseDataset(dataframe, sensors=None, label_columns='activity code', as_array=True)[source]

Bases: librep.utils.dataset.PandasDataset

An abstract class representing a generic map-style dataset which implement the getitem and len protocols. All datasets that represent a map from keys to data samples should subclass it.

Datasets subclassed from this class can be acessed using the subscription syntax, such as dataset[index]. It usually return a tuple where the first element represent the sample and the second, the label.

All map-style datasets must be implement the len protocol which returns the number of samples in the dataset.
