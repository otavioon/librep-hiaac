Spectrogram

For filtering or modifying the data, in a supervised or unsupervised way.
class librep.transforms.spectrogram.Spectrogram(fs=100, segment_size=50, overlap=30)

Bases: librep.base.transform.Transform

For filtering or modifying the data, in a supervised or unsupervised way. fit allows implementing parametrizable transforms. This method sees the whole dataset. transform allows transforming each sample.
